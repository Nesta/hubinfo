HubInfo

HubInfo is a jQuery plugin to show information about your GitHub repo. Here is an example:

Download

Usage

Include hubinfo.js and hubinfo.css on your page and then put this in a script tag: 

$('#hubInfo').hubInfo({ 
    user: 'jgallen23',
    repo: 'hubinfo'
});
